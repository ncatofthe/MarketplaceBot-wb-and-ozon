
Инструкция по возможным ошибкам при работе бота
===============================================

ОШИБКИ ПОДКЛЮЧЕНИЯ К API
--------------------------

1. "Ozon: API ключ или Company ID не настроены"
   Причина: не заполнены поля API Key или Company ID в настройках
   Решение: введите корректные данные в настройках бота

2. "Wildberries: API ключ не настроен"
   Причина: не заполнено поле API Key для Wildberries
   Решение: введите API ключ в настройках

3. "Ozon: Не удалось выполнить тестовый запрос к API"
   Причина: неверные учетные данные или проблемы с сетью
   Решение: проверьте правильность API Key и Company ID

4. "WB API: Ошибка авторизации. Проверьте API ключ"
   Причина: неверный API ключ Wildberries
   Решение: получите новый ключ в личном кабинете WB

5. "Получен 403 (антибот защита)"
   Причина: Ozon заблокировал запросы из-за частых обращений
   Решение: увеличьте задержку между запросами в настройках

ОШИБКИ ПОЛУЧЕНИЯ ОТЗЫВОВ
--------------------------

6. "Ozon: Ошибка при получении отзывов"
   Причина: проблема с соединением или API Ozon недоступен
   Решение: проверьте интернет-соединение, попробуйте позже

7. "Wildberries: Ошибка при получении отзывов"
   Причина: проблема с соединением или API WB недоступен
   Решение: проверьте интернет-соединение, попробуйте позже

ОШИБКИ ОТПРАВКИ ОТВЕТОВ
-------------------------

8. "Ozon: Не удалось отправить ответ на отзыв"
   Причина: API вернул ошибку
   Решение: проверьте логи для деталей, возможно проблема с модерацией

9. "WB API: Не удалось отправить ответ на отзыв"
   Причина: API вернул ошибку
   Решение: проверьте логи, возможно ответ слишком длинный

10. "Ответ не отправлен - возможно отзыв уже отвечен"
    Причина: бот пытается ответить на уже отвеченный отзыв
    Решение: это нормальное поведение, бот продолжит работу

ОШИБКИ НАСТРОЕК
-----------------

11. "Интервал проверки слишком маленький"
    Причина: установлено значение меньше 1 минуты
    Рекомендация: установите интервал не менее 5 минут

12. "min_stars больше 5"
    Причина: некорректное значение в настройках
    Решение: установите значение от 1 до 5

13. "Шаблоны ответов пустые"
    Причина: файл шаблонов поврежден или пуст
    Решение: нажмите "Сбросить по умолчанию" в разделе шаблонов

14. "Ошибка JSON при сохранении шаблонов"
    Причина: неверный формат JSON в редакторе
    Решение: проверьте синтаксис JSON, не удаляйте кавычки и скобки

ОШИБКИ ЗАПУСКА
---------------

15. "Бот уже запущен"
    Причина: попытка запустить бота повторно
    Решение: сначала остановите бота, затем запустите снова

16. "Нет модуля named 'requests'"
    Причина: не установлены зависимости
    Решение: выполните pip install -r requirements.txt

17. "No module named 'tkinter'"
    Причина: tkinter не установлен в системе
    Решение: установите tkinter (в Linux: sudo apt-get install python3-tk)

18. "PermissionError при записи логов"
    Причина: нет прав на запись в папку logs
    Решение: запустите программу от администратора или проверьте права папки

19. "Config файл поврежден"
    Причина: файл settings/config.local.json имеет неверный формат
    Решение: удалите файл и перезапустите бота, затем заново сохраните настройки в GUI

ОШИБКИ API OZON
-----------------

20. "antibot protection"
    Причина: слишком частые запросы к API Ozon
    Решение: увеличьте "Задержка между запросами" до 1-2 секунд

21. "error: 'limit' is required"
    Причина: проблема с форматом запроса к API
    Решение: переустановите бота, возможно повреждение файлов

22. "error: company_id is required"
    Причина: не указан Company ID
    Решение: укажите Company ID в настройках

ОШИБКИ API WILDBERRIES
-----------------------

23. "error: 'limit' must be integer"
    Причина: неверный тип параметра limit
    Решение: переустановите бота

24. "Suppliers API is deprecated"
    Причина: API WB изменился
    Решение: проверьте актуальность версии бота

25. "feedback not found"
    Причина: отзыв был удален или недоступен
    Решение: бот продолжит работу со следующими отзывами

ОШИБКИ ПОТОКОВ
---------------

26. "Thread join timeout"
    Причина: поток не остановился за 10 секунд
    Решение: это не критично, бот остановится принудительно

27. "RuntimeError: main thread is not in main loop"
    Причина: проблема с GUI потоком
    Решение: перезапустите приложение

ОШИБКИ КОНФИГУРАЦИИ
--------------------

28. "Answers file not found"
    Причина: файл шаблонов ответов отсутствует
    Решение: перезапустите бота, файл создастся автоматически

29. "Invalid JSON in answers file"
    Причина: файл шаблонов содержит ошибки JSON
    Решение: нажмите "Сбросить по умолчанию" в разделе шаблонов

30. "Log file cannot be created"
    Причина: нет места на диске или прав на запись
    Решение: очистите папку logs или измените права доступа

ОШИБКИ EXE ФАЙЛА
------------------

31. "При запуске exe ничего не происходит"
    Причина: отсутствуют библиотеки или несовместимость
    Решение: проверьте что файл MarketplaceBot.exe присутствует в папке dist/MarketplaceBot/

32. "Ошибка при запуске exe на Windows 10"
    Причина: могут отсутствовать Visual C++ Redistributable
    Решение: установите Visual C++ Redistributable

33. "EXE файл поврежден"
    Причина: при сборке что-то пошло не так
    Решение: пересоберите exe используя build.bat или python -m PyInstaller MarketplaceBot.spec

ЧАСТО РЕШАЕМЫЕ ПРОБЛЕМЫ
-------------------------

Если бот не работает:

1. Проверьте что API ключи введены правильно
2. Проверьте подключение к интернету
3. Убедитесь что бот включен (галочка в настройках)
4. Проверьте что интервал проверки больше 0
5. Перезапустите приложение
6. Проверьте логи во вкладке "Логи"

Если ответы не отправляются:

1. Проверьте что отзыв не был уже отвечен
2. Проверьте что min_stars соответствует оценке отзыва (по умолчанию 1)
3. Проверьте что в настройках установлен флаг "enabled"
4. Проверьте логи на наличие ошибок API

Если бот падает с ошибкой:

1. Запишите текст ошибки из логов
2. Перезапустите приложение
3. Проверьте что установлены все зависимости
4. Обратитесь за помощью с текстом ошибки

## КАК ПЕРЕСОБРАТЬ EXE

Если вам нужно пересобрать exe файл:

1. Откройте командную строку в папке проекта
2. Выполните: `python -m pip install -r requirements-build.txt`
3. Выполните: `build.bat`
4. Готовый exe появится в папке `dist/MarketplaceBot/`

Альтернатива вручную: `python -m PyInstaller --clean --noconfirm MarketplaceBot.spec`
